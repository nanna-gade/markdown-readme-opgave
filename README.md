# Markdown Dokumentation

## Fordele ved at bruge Markdown:
1. Høj læsbarhed: Teksten forbliver læselig i sin rå form, da syntaksen er minimalistisk og intuitiv.

2. Universel kompatibilitet: Filerne gemmes som .md (ren tekst), hvilket betyder, at de kan åbnes i alle teksteditorer på alle operativsystemer.

3. Effektivitet: Formatering udføres direkte med tastaturet uden brug af mus eller grafiske menuer, hvilket bevarer skriverens fokus.

4. Versionsstyring: Markdown er velegnet til systemer som Git, da ændringer kan spores præcist linje for linje.

5. Nem konvertering: Dokumenter kan hurtigt eksporteres til HTML, PDF eller Word-formater via værktøjer som Pandoc.

6. Fremtidssikring: Da formatet ikke er bundet til en specifik softwareproducent, vil filerne forblive læsbare uanset fremtidige programopdateringer.
